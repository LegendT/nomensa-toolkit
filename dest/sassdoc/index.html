    grid-width

    @function grid-width($columns) { ... }
    View source

    Description

    Get width of grid item calc() is returned b/c you can use px gutters and % width.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $columns

    How many columns from $grid-columns (i.e. 12) to get width for. Passing in 4 when 12 column would get you roughly 25% after accounting for gutters.

    		— none

    Returns

    Number —

    a calc() value to use in width

    Example

    .class {
  width: grid-width(4); // becomes `width: calc((50% + 1%) - 2%);`
}

    Requires

    • [variable] grid-columns

    Used by

    • [mixin] smart-grid
